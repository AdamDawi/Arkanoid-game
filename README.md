# Arkanoid-game
This is retro game named arkanoid made in c++ builder.

### Getting started:
Player have paddle to bounces a ball into the blocks on the top of the screen. Game ends when player destroy all blocks.

### Features:
- control with arrows
- c++ builder timer game
